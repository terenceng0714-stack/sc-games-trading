# SC Games Website

Modern responsive website for a premium gaming console micro-repair specialist
at Holiday Plaza, Johor Bahru.

## Tech Stack

- Next.js App Router
- React
- TailwindCSS
- TypeScript
- Vercel-ready configuration

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Customize

- WhatsApp number and shared site data: `components/site-data.ts`
- Home page: `app/page.tsx`
- Services page: `app/services/page.tsx`
- Pricing page: `app/pricing/page.tsx`
- Booking page: `app/booking/page.tsx`
- About page: `app/about/page.tsx`

Replace the placeholder WhatsApp number in `components/site-data.ts` before
deploying.
